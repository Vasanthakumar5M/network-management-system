Android Certificate Installation

Follow these steps to install the security certificate on your Android device

1 Download
2 Open Settings
3 Install
4 Complete

Step 1: Download Certificate

Tap the button below to download the security certificate.

Download Certificate
The file will be saved to your Downloads folder.

Step 2: Open Security Settings

Choose your Android version:

  1. Settings > Security > More Security Settings

    Or search for "Install certificates" in Settings

  2. Tap "Encryption & credentials"
  3. Tap "Install a certificate" > "CA certificate"
  1. Settings > Security > Advanced
  2. Tap "Encryption & credentials"
  3. Tap "Install from storage"
  1. Settings > Security & Location
  2. Tap "Advanced" > "Encryption & credentials"
  3. Tap "Install from SD card"

Step 3: Install the Certificate

  1. Select the downloaded certificate

    Navigate to Downloads and select "network-security.cer"

  2. Read the warning and tap "Install anyway"

    Android will show a security warning - this is normal

  3. Enter your PIN or pattern

    Verify with your screen lock

  4. Name the certificate

    Enter "Network Security" or any name you prefer

  5. Select "VPN and apps" for credential use

    Then tap OK or Done

You may see a notification that your network may be monitored. This is expected.

Installation Complete!

You're all set!

The security certificate has been successfully installed on your Android device.
