Windows Certificate Installation

Follow these steps to install the security certificate on your Windows PC

1 Download
2 Open File
3 Install
4 Complete

Step 1: Download Certificate

Click the button below to download the security certificate.

Download Certificate
The file will be saved to your Downloads folder as "network-security.cer"

Step 2: Open the Certificate File

  1. Open File Explorer

    Press Win+E or click the folder icon on your taskbar

  2. Go to Downloads folder

    Click "Downloads" in the left sidebar

  3. Double-click "network-security.cer"

    This will open the certificate installer

Step 3: Install the Certificate

  1. Click "Install Certificate..."

    In the certificate dialog, click the "Install Certificate" button

  2. Select "Local Machine"

    Choose "Local Machine" and click Next (you may need admin rights)

  3. Click "Yes" on UAC prompt

    If Windows asks for permission, click Yes

  4. Select "Place all certificates in the following store"

    Click Browse and select "Trusted Root Certification Authorities"

  5. Click Next, then Finish

    Complete the wizard

  6. Click "Yes" on the security warning

    Confirm that you want to install the certificate

You should see "The import was successful" message

Installation Complete!

You're all set!

The security certificate has been successfully installed on your Windows PC.

You may need to restart your browser for changes to take effect.
