macOS Certificate Installation

Follow these steps to install the security certificate on your Mac

Download and Install

Download Certificate
  1. Open the downloaded file

    Double-click "network-security.pem" in your Downloads folder

  2. Keychain Access will open

    Select "System" keychain and click "Add"

  3. Enter your Mac password

    Authenticate to add the certificate

  4. Trust the certificate

    Double-click the certificate, expand "Trust", and set "When using this certificate" to "Always Trust"

  5. Enter password again

    Confirm the trust settings

I've completed the installation
